Blog Update II

By Gabriel on 2019-08-29

Again it’s been quiet a while since updating the blog. Because of this and to prevent people think the game is abandoned, I added three new things to the website. Let’s see them!

Latest commit bar

On the top you will see a new gold bar with the last time I made a commit to the code base. I will try to write meaningful descriptions from now on!

Road map progress

This is a biggie that has it’s own post here. Basically gives you and idea of what brings the future to the project up to the point the Beta is released with a shadowed progress bar. Each section links to the corresponding paragraph in the road map.

Ribbon

Finally, the ribbon, aimed to direct traffic towards the mailing list. Be sure to subscribe if you want to be one of the first to receive an invite for the Alpha!
